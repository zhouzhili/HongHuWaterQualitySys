# 洪湖水质反演系统
<ul>
<li>
使用C#语言，基于ArcGIS Engine开发反演信息系统，采用SQL Server作为数据库，管理历年来的影像数据信息。
</li>
<li>
该平台包含影像数据管理及辐射定标、大气校正等基本预处理流程，并结合矢量数据信息，可反演洪湖整个湖区的水质分布情况，并可自动生成水质分布专题图，完成了水质反演到出图的自动化处理。
</li>
</ul>
